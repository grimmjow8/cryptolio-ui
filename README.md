# Cryptocurrency Portfolio

**Application to view management information for cryptocurrencies.**


## To Use

To clone and run this repository you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/). Run the following:

```bash
# Clone this repository
git clone https://github.com/grimmjow8/crypto-management
# Go into the repository
cd crypot-ui
# Install dependencies
npm install
# Run the app
npm start
# Or run
./start.sh
```
